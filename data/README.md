# data/README.md — Data sources and preprocessing

## Synthetic graphs (generated at runtime)

All Forest-Fire and Modular graphs are generated programmatically — no download needed.

```python
from src.env.graph_generators import generate_forest_fire
g = generate_forest_fire(n=1000, p=0.37, pb=0.32, seed=42)
```

Parameters `p=0.37, pb=0.32` match the AAAI 2027 paper and are set in all
`configs/base_config.yaml` and experiment configs.

## Rice Facebook Network (real-world topology)

### Source and License

The data is from the **Facebook100** collection:

> Traud, A.L., Kelsic, E.D., Mucha, P.J. & Porter, M.A. (2012).
> "Comparing Community Structure to Characteristics in Online Collegiate
> Social Networks." *SIAM Review*, 54(3):526–543.
> DOI: 10.1137/080734030

The Facebook100 dataset contains anonymised snapshots of 100 U.S. university
Facebook networks collected in September 2005. No personally identifiable
information is included (only status, graduation year, and dorm).

**Redistribution notes:**

### Preprocessed graph (generated locally, NOT in git)
```
data/graphs/rice_facebook.pkl   ← NetworkX DiGraph, largest weakly connected component
```

### Regenerating the pickle
```python
# experiments/download_networks.py handles this:
python experiments/download_networks.py
```
Or manually:
```python
import networkx as nx, pickle
G = nx.read_edgelist('data/raw/rice-facebook-undergrads-links.txt',
                     create_using=nx.DiGraph(), nodetype=int)
G = G.subgraph(max(nx.weakly_connected_components(G), key=len)).copy()
G = nx.convert_node_labels_to_integers(G)
with open('data/graphs/rice_facebook.pkl', 'wb') as f:
    pickle.dump(G, f)
print(f"Rice-FB: n={G.number_of_nodes()}, m={G.number_of_edges()}")
```

### Expected statistics
- n ≈ 217 nodes (largest WCC of the Rice network)
- m ≈ 3,000–4,000 directed edges
- Used in zero-shot generalization experiments (OOD topology)

### Loader path
`src/env/graph_generators.py → load_rice_facebook(path='data/graphs/rice_facebook.pkl')`
